Rollback (if something breaks)

  1. MINI: for f in ~/Library/LaunchAgents/com.daisy.*.plist; do launchctl unload "$f"; done
  2. MINI: tmux kill-session -t daisy
  3. MBP: for f in ~/Library/LaunchAgents/com.daisy.*.plist; do launchctl load "$f"; done
  4. MBP: Restart Daisy's tmux session (whatever you normally do)
  5. Send a test Telegram message — reply should come from MacBook again

Total time to rollback: ~5 minutes. The MacBook is unchanged — files and configs are intact.

When the dust settles: let things run for a week, then come back to plan the MacBook clean-up (or repurpose).
